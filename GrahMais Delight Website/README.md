GrahMais Delight Responsive Website

Project Description

GrahMais Delight Responsive Website is a 4-page website for a small dessert company. The site displays the business introduction, product highlights, business info, owner profile, values and contact.

The website is responsive for mobile, tablet and desktop browsers with the help of semantic HTML5, external CSS, Flexbox, CSS Grid and media queries.

Project Goal

This project is about creating a well-structured, responsive and clean HTML5 and CSS3 website. It also showcases good navigation, semantic structure, responsiveness and a fully functional contact form.

Pages Included

- `Home.html` includes introduction, hero section, dessert image and business highlights.
- `About.html` - contains the business description, owner section, and business values.
- `Contact.html` - contains the contact form and contact details.
- `style.css` - contains all the external CSS styles used in the website.

Features

- Multi-page website with 3 pages
- Working navigation links between pages
- Semantic HTML5 elements like header, nav, main, section, article, aside and footer.
- External CSS file for all styles
- No inline CSS styling.
- Flexbox navigation bar
- Customizable card and page sections with CSS Grid layout
- Mobile-first responsive design
- Media queries for tablet and desktop screens
- Functional-looking contact form
- Correct form label with references to inputs
- Accessible images with descriptive alt text
- Clean and commented code

Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
